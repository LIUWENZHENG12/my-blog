📄 使用說明 - 資訊技師考前網站

1️⃣ 開啟方式
------------------
開啟 index.html 即可使用（雙擊或用瀏覽器打開）
手機、平板、電腦皆可自動調整版面。

2️⃣ 功能介紹
------------------
✅ Chatbot 練題模式：
  - 輸入「幫我抽題」隨機出一題。
  - 可輸入關鍵字：
      OSI七層、ACID、SDLC、CIA、Deadlock、Big O、SQL、Normalization、TCP/IP、技師法。

✅ 考前重點區塊：
  - 位於網頁導覽列「考前重點」中，包含10天背誦表。

3️⃣ 客製化
------------------
📂 可在 images 資料夾放入自己的照片。
📄 可自行修改 index.html 或 chatbot.js 新增題目。

祝你考試順利，保持穩定節奏就能拿分！
